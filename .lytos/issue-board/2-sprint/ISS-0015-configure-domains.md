# ISS-0015 — Configure lytos.org and lytos.dev domains

## What to do

1. Point `lytos.org` to the website (Cloudflare Pages or GitHub Pages)
2. Configure `lytos.dev` as redirect to `lytos.org` (or reserve for future technical use)
3. Update DNS records
4. Configure HTTPS

## Definition of done

- `lytos.org` serves the documentation website
- `lytos.dev` is configured (redirect or reserved)
- HTTPS works on both domains
